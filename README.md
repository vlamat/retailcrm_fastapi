# RetailCRM FastAPI Test Project

## Описание
REST API для работы с RetailCRM через FastAPI.

## Запуск проекта

1. Создайте `.env`:

```bash
RETAILCRM_API_URL=Ваш адресс
RETAILCRM_API_KEY=Ваш API
```

2. Запустите:

```bash
docker-compose up --build
```

3. Откройте в браузере:

```
http://localhost:8000/docs
```
